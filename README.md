🪵 ShopBot Custom Furniture Generator

A web-based tool to generate customizable furniture designs for CNC machines such as ShopBot.

This project allows users to adjust parameters and generate ready-to-cut SVG data for furniture like chairs, desks, shelves, and stools.

✨ Features
Parametric furniture design
Real-time preview using p5.js
SVG export for CNC / laser cutting
Multiple furniture types:
Chair
Desk
Shelf
Stool
Adjustable parameters:
Dimensions (width, depth, height)
Material thickness
Structural variations
🛠️ Tech Stack
p5.js
p5.js-svg
HTML / CSS / JavaScript
🚀 Getting Started
Clone this repository
git clone https://github.com/your-username/shopbot-custom-furniture-generator.git
Open the project
cd shopbot-custom-furniture-generator
Run locally

Just open index.html in your browser.

📐 How It Works

Each furniture type is generated using parametric geometry.

Users control parameters via sliders, which update:

Shape
Dimensions
Joinery structure

The system converts these into 2D layouts suitable for CNC cutting.

📦 Export

Click the Download SVG button to export the design.

The output file can be used for:

ShopBot
Laser cutting
CNC machining
